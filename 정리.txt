spa
SPA (Single Page Application)는 웹 애플리케이션의 한 종류로, 전통적인 다중 페이지 웹 애플리케이션과 달리 하나의 HTML 페이지로 시작하고 JavaScript를 사용하여 페이지 갱신을 처리
SPA는 새로운 페이지를 로드하지 않고도 사용자 경험을 개선할 수 있습니다. 
예를 들어, 사용자가 메뉴에서 다른 항목을 선택하면, SPA는 페이지 전체를 새로 고치지 않고 필요한 부분만 업데이트하며 변경된 내용을 표시합니다. 
이것은 일반적으로 더 나은 반응성, 더 빠른 속도 및 더 뛰어난 사용자 경험을 제공합니다.
하지만 SPA가 웹을 앱처럼 사용하는 것은 아닙니다. 
SPA는 웹 기술을 사용하여 구축되며, 대부분의 경우에는 브라우저에서 실행됩니다. 반면에, 네이티브 앱은 모바일 운영체제에 특화된 프로그램으로, 전용 언어나 도구를 사용하여 개발됩니다.
